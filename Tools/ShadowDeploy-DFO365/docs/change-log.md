# [1.0.0] - Initial Release

## 🎉 Initial Public Release

Shadow Deploy – Defender for Office 365 is the Email Gatekeeper of the Shadow Suite, providing a guided deployment, scoping, validation, and reporting experience for Microsoft Defender for Office 365.

## Known Limitations

* Designed for Windows PowerShell environments
* Requires ExchangeOnlineManagement module
* Requires appropriate Microsoft 365 administrative permissions
* Validation scenarios are advisory and should be executed within approved testing processes
